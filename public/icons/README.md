# PWA Icons Required

You need to add the following icon files to this directory:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

These should be square PNG images with the VB Cart logo.

You can use online tools like:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/
- https://favicon.io/

Or use your existing logo at /public/logos/vb_logo.png as a base and resize it to these dimensions.
